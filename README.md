# dingzehu.github.io

Personal portfolio site — static HTML, CSS, and JavaScript — hosted with [GitHub Pages](https://pages.github.com/).

## Live site

**[https://dingzehu.github.io/](https://dingzehu.github.io/)**

## Contents

The homepage (`index.html`) includes professional experience, about, technical skills, projects with links to GitHub repositories, qualifications, and certifications.

## Local preview

Clone the repository and open `index.html` in a browser, or serve the folder with any static file server, for example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy

Pushing to the default branch of this repository updates the live site automatically (GitHub Pages for `username.github.io` user sites).
